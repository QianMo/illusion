# ILLUSION
**illusion**是一个使用[Protobuf](https://developers.google.com/protocol-buffers/)（Google  Protocol Buffers）文件作为配置描述，然后读取EXCEL文件里面相关的配置，然后生产配置数据文件的工具。主要用于游戏配置管理等。
